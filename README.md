# Where Are My Keys?

## Descripción

`Where Are My Keys?` es un prototipo web pensado para ayudar a recordar dónde se han dejado las llaves dentro de casa. El usuario puede marcar una zona del plano de su vivienda y consultar rápidamente la última ubicación registrada.

Está dirigido a cualquier persona que quiera evitar perder tiempo buscando sus llaves antes de salir. Además, incluye una vista de emergencia con contactos de apoyo para situaciones en las que no se pueden recuperar de forma inmediata.

El proyecto cubre una necesidad cotidiana: reducir olvidos y mejorar la organización personal mediante una interfaz simple, visual y accesible tanto en móvil como en escritorio.

## Estructura del Prototipo

El prototipo tiene tres vistas conectadas por una barra de navegación superior presente en todas las páginas.

- **index.html:** Vista principal de localización. Incluye un plano interactivo para marcar dónde están las llaves y una sección de ubicaciones recientes.
- **crear-planos.html:** Vista de registro de vivienda. Permite introducir datos básicos de la propiedad y subir un plano.
- **emergencia.html:** Vista de apoyo en caso de urgencia. Muestra contactos de respaldo y un formulario para reportar una emergencia.

## Tecnologías Utilizadas

- HTML5
- CSS3
